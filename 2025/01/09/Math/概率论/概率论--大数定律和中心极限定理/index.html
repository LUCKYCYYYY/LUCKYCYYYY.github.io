# 切比雪夫不等式

# 辛钦大数定律

设X1,X2,...,Xn...X_1,X_2,...,X_n...X1​,X2​,...,Xn​... 是独立同分布的随机变量序列，若期望EXn=μEX_n=\muEXn​=μ 有限，则对任意正数ε>0\varepsilon>0ε>0，有

lim⁡n→∞P(∣Xn‾−μ∣<ε)=1\lim_{n\to\infty}P(|\overline{X_n}-\mu|<\varepsilon)=1 n→∞lim​P(∣Xn​​−μ∣<ε)=1

# 伯努利大数定律
