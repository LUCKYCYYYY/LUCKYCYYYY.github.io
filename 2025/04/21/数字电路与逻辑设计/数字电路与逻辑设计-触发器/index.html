# 触发器概述

触发器是具有记忆功能的逻辑器件

  1. 有两个互补的输出端 QQQ 和 Qˉ\bar QQˉ​
  2. 有两个稳定状态 000 和 111
    • 输出端 Q=1Q=1Q=1，称为 “1” 状态，也称为置位状态 Set
    • 输出端 Q=0Q=0Q=0，称为 ”0“ 状态，也称为复位状态 Reset
  3. 触发器能够接收，保存和输出信号
    • 输入信号不发生变化时，触发器状态稳定不变
    • 在一定输入信号作用下，触发器可以从一个稳定状态到另一个稳定状态；输入信号撤销后，保持新的状态不变
    • 输入信号作用之前的状态称为触发器的现态，记为 QnQ^nQn，简写为 QQQ；作用之后的状态称为次态，记作 Q^

# 与非门构成

当输入 R=0R=0R=0，与任何异或取反之后都为 111，所以 Qˉ=1\bar Q=1Qˉ​=1；反馈回的输入为 111

RRR 和 SSS 都是低电平有效

当 R=0R=0R=0 时，置 000；当 S=0S=0S=0 时，置 111

次态方程：

Qn+1=Sˉ+R⋅QQ^{n+1}=\bar S+R\cdot Q Qn+1=Sˉ+R⋅Q

约束方程：

R+S=1R+S=1 R+S=1

# 或非门构成

次态方程：

Qn+1=S+RˉQQ^{n+1}=S+\bar RQ Qn+1=S+RˉQ

约束方程：

R⋅S=0R\cdot S=0 R⋅S=0

# 空翻

同一个时脉冲作用期间触发器状态发生两次或两次以上变化的现象

产生原因：

  1. 时钟脉冲作用期间，输入信号直接控制着触发器状态的变化
    • CPCPCP 为 1 时，输入信号发生变化，触发器状态会跟着变化
  2. 时钟宽度控制不够
    • CPCPCP 为 1 时间过长，输入的多次变化得到完全响应，使得一个时钟脉冲作用期间触发器多次翻转

“空翻” 将造成状态的不确定和系统工作的混乱，这是不允许的。
