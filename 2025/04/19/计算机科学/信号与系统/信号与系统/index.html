# 基本系统性质

# 记忆性

无记忆系统：如果一个系统的输出只取决于该时刻的输入

# 可逆性

一个系统如果在不同输入下，导致不同输出，则该系统为可逆系统

如果一个系统是可逆系统，那么它一定存在逆系统

当逆系统和原系统级联后，就会产生一个输出 w[n]w[n]w[n] 等于第一个系统的输入 x[n]x[n]x[n]

# 因果性

如果一个系统在任何时刻的输入只取决于现在的输入及过去的输入

示例

y[n]=x[−n]y[n]=x[-n] y[n]=x[−n]

不是因果的。

因为 y[−4]=x[4]y[-4]=x[4]y[−4]=x[4] 用到了将来的值 x[4]x[4]x[4]

y(t)=x(t)cos⁡(t)y(t)=x(t)\cos(t) y(t)=x(t)cos(t)

是因果的，而且是无记忆的。

因为这个系统中任何时刻 ttt 的输出等于在同一时刻的输入再乘以一个随之间变化的数，我们可以将它写成

y(t)=x(t)g(t)y(t)=x(t)g(t) y(t)=x(t)g(t)

# 稳定性

一个稳定的系统，如果输入是有界的（即输入的幅度不是无界增长的），则系统的输出也必须是有界的，因此不可能发散

# 时不变性

若系统的特性和行为不随时间而变，该系统就是时不变的

如果用信号与系统的语言描述：如果在输入信号上有一个时移，而在输出信号中产生同样的时移，那么这个系统就是时不变系统

即当输入为 x[n−n0]x[n-n_0]x[n−n0​] 时，输出是 y[n−n0]y[n-n_0]y[n−n0​] ，那么该系统是时不变系统

示例

y[n]=nx[n]y[n]=nx[n] y[n]=nx[n]

是时变系统。

因为当输入 x[n−n0]x[n-n_0]x[n−n0​] 时

输出=nx[n−n0]输出=nx[n-n_0] 输出=nx[n−n0​]

与

y[n−n0]=(n−n0)x[n−n0]y[n-n_0]=(n-n_0)x[n-n_0] y[n−n0​]=(n−n0​)x[n−n0​]

# 线性

如果某一个输入是由几个信号的加权和组成的，那么输出也就是系统对这组信号中每一个响应的加权和

令 y1(t)y_1(t)y1​(t) 是一个连续时间系统对输入 x1(t)x_1(t)x1​(t) 的响应，而 y2(t)y_2(t)y2​(t) 是对于输入 x2(t)x_2(t)x2​(t) 的输出

可加性：y1(t)+y2(t)y_1(t)+y_2(t)y1​(t)+y2​(t) 是对 x1(t)+x2(t)x_1(t)+x_2(t)x1​(t)+x2​(t) 的响应

比例性 / 齐次性：ay1(t)ay_1(t)ay1​(t) 是对 ax1(t)ax_1(t)ax1​(t) 的响应

一个系统是线性的，必须同时满足可加性和比例性
