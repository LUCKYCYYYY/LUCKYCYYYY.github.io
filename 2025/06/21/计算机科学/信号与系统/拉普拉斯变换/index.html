# 拉普拉斯变换

在一个冲激响应为 h(t)h(t)h(t) 的线性时不变系统，对 este^{st}est 复指数输入信号的响应 y(t)y(t)y(t) 是

y(t)=H(s)esty(t)=H(s)e^{st} y(t)=H(s)est

其中

H(s)=\int_{-\infty}^{+\infty}h(t)e^{-st}dt--❤️
  • 如果 sss 为纯虚数，即 s=jws=jws=jw，那么式 ❤️ 对应的就是傅里叶变换
  • 如果 sss 为一般的复变量，那么式 ❤️ 就称为单位冲激响应 h(t)h(t)h(t) 的拉普拉斯变换

一个信号 x(t)x(t)x(t) 的拉普拉斯变换定义为

X(s)≜∫−∞∞x(t)e−stdtX(s) \triangleq\int_{-\infty}^{\infty}x(t)e^{-st}dt X(s)≜∫−∞∞​x(t)e−stdt

注意这是一个自变量为 sss 的函数，复变量 sss 可以写成 s=σ+jws=\sigma+jws=σ+jw

为了方便将拉普拉斯变换表示成为算子 L(x(t))\mathcal{L}(x(t))L(x(t))

x(t)x(t)x(t) 和 X(s)X(s)X(s) 之间的关系记为

x(t)↔LX(s)x(t)\xleftrightarrow{\mathcal L} X(s) x(t)L​X(s)
