交换瓶子

在a[i] = x && a[x] != i的情况下，需要找到一些“圈”，每个圈中的数据可以通过交换回到原来的位置（即所需的数圈中都包含），而交换的次数应该是圈的大小-1

蚂蚁感冒

  1. 将碰撞的蚂蚁掉头看做是两只蚂蚁不掉头，都向前走，只感染
  2. 分析得：对于初始感染蚂蚁a，在a左边并向右走的蚂蚁可能感染，在a右边向左走的蚂蚁可能感染
  3. 特殊情况：a左边无向右走的蚂蚁，a右边无向左走的蚂蚁\(\rightarrow\)不会感染别的蚂蚁

修改数组

优化：a[i] += (b[a[i]] - 1)，b数组储存的是经由i晋升的数的所有个数，像跳跃一样告诉你至少要跳多远，直到你跳到一个无人之境

连号区间数

因为题目给定的数组是1 ~ n的排列

所以所有的数都是不重复的，如果一个区间的最大值-最小值和区间长度相同，就说明可以构成连续子区间

前缀和

k倍区间

优化：

用数组c[N]记录之前有相同余数的前缀和，因为除以k余数相同的两个数相减一定可以被k整除，所以用当前这个前缀和减去之前有相同余数的前缀和一定可以被k整除，所以res += c[a[i] % k]表示加上之前有相同余数的前缀和的数量

初始化c[0] = 1，因为余数为0的时候可以被k整除，如果不初始化为0，那么遇到a[i]能被整除的情况相当于res += 0，错误

日期问题

  1. 只有三种排序，不是随意排的
  2. 要按照从小到大排
  3. 注意闰年的情况，每个月月份天数不同

  1. 遍历数组b
  2. upper_bound和lower_bound的使用

Fibonacci 数列与黄金分割

交换次数

  1. 枚举所有可能的顺序，计算达成该结果所需的交换次数，比较得到最小值
  2. 对于BAT，计算字符串s中B和T的总个数，以及已经在正确位置上的字母的个数，二者相加，减去可以BT互换的次数既是最终次数

整理玩具

  1. 利用map，map<char, vector<pair<int, int>>>记录每个数出现的坐标
  2. 计算每个矩形的大小是否等于该数的个数

二进制

1~n随机选取

利用二进制数进行状态压缩

题目要求的结果是\(2^n\)个，这\(2^n\)个选择情况，对应于一个n位的2进制数的各个位取0或取1的情况

例如：n=3，即 000 -> \n 001 -> 1 010 -> 2 100 -> 3 011 -> 1 2 101 -> 1 3 110 -> 2 3 111 -> 1 2 3

机器人塔

  1. 利用二进制数枚举最后一层的情况
  2. 利用bitset能轻松算出某个数的二进制有多少个1
  3. x ^= (x >> 1)相当于和旁边的数异或，A是0，B是1，则A ^ A = A, B ^ B = A, A ^ B = B, B ^ A = B与题目要求相同
  4. x &= (1 << (i - 1)) - 1是为了只保留i - 1个数，例如i = 4则xx &= 0111

拼十字

  • 先按照长从大到小排序

  • 因为树状数组要算的是在此之前宽度小于该宽度的个数，所以如果在同一个长度下，按照宽度从小到大排序就会把长度相同但是宽度更小的矩形算入答案中，与题目的长度严格大于不符，所以宽度也从大到小
