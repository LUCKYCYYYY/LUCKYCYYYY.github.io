# 时序逻辑电路概述

输入信号：X1∼XnX_1\sim X_nX1​∼Xn​

输出信号：Z1∼ZmZ_1\sim Z_mZ1​∼Zm​

内部输入信号：y1∼ysy_1\sim y_sy1​∼ys​

时序逻辑电路中的激励信号：Y1∼YrY_1\sim Y_rY1​∼Yr​

按照是否存在同一的时钟信号：

  1. 同步时序逻辑电路
  2. 异步时序逻辑电路

按照输入信号的不同，脉冲异步时序可分为：

  1. 脉冲异步时序逻辑电路
  2. 电平异步时序逻辑电路

脉冲的分类：

  • 正脉冲：信号的初始值是低电平
  • 负脉冲：信号的初始值是高电平
  • 周期性的时钟信号：脉冲信号之间的间隔时间是固定的
  • 电平信号：如果信号不具备周期性，就得到了电平信号

根据输出信号的特点：

  • MealyMealyMealy 型：输出不仅取决于当前电路状态，还同时与电路输入有关
  • MooreMooreMoore 型：输出只取决于当前电路的状态

同步时序逻辑电路和脉冲异步时序逻辑电路的共同特点：

  1. 电路状态时在脉冲下实现的
    • 同步时序逻辑电路：尽管输入信号可以是电平信号或者脉冲信号，但电路的转换受统一的时钟脉冲控制
    • 脉冲异步时序逻辑电路：没有统一的时钟脉冲，输入信号为脉冲信号，即控制电路状态转换的脉冲信号时由电路输入端提供的
  2. 电路对过去输入信号的记忆是由触发器实现的

# 小规模同步时序逻辑电路的分析

通常将时钟脉冲信号 CPCPCP 当作一种默认的时间基准，在写激励函数表达式时，可以不用列出时钟脉冲信号 CPCPCP 的表达式

# 小规模脉冲异步时序逻辑电路的分析

在脉冲异步时序逻辑电路中，触发器不共用一个时钟脉冲信号，引起触发器状态变化的脉冲信号是由输入端直接提供的

MealyMealyMealy 型电路的输出不仅是状态变量的函数，而且是输入的函数，所以输出是脉冲信号

MooreMooreMoore 型电路的输出仅仅是状态变量的函数，输出信号的变化只能发生在两个间隔不定的输入脉冲之间，所输出是电平信号

对输入脉冲信号的约束条件：

  • 输入脉冲的宽度，必须保证触发器可靠翻转
  • 输入脉冲的间隔，必须保证钱一个脉冲引起的电路响应完全后，后一个脉冲才能到来
  • 不允许两个或两个以上输入端同时出现脉冲，所以分析时只需考虑各自单独出现脉冲的 nnn 种情况

# 小规模脉冲异步时序逻辑电路的设计

  1. 由于不允许两个或两个以上输入端同时出现脉冲

    • 在形成原始状态图和原始状态表时，若有多个输入信号，只需考虑多个输入信号中仅一个出现脉冲的情况
    • 在确定激励函数和输出函数时，可将两个或两个以上输入同时出现脉冲的情况作为无关条件处理

  2. 由于电路中没有统一的时钟脉冲，当存储电路采用带时钟控制的触发器时，触发器的时钟端必须作为激励函数处理

    当触发器状态保持不变时，激励有两种取值：（往往选择 CP=0CP=0CP=0）

    • 一种是令 CPCPCP 端为 ddd，输入端取相应值
    • 一种是令 CP=0CP=0CP=0，激励端取任意值

  3. 一般考虑使用输入脉冲后沿触发的触发器（正脉冲的后沿是下降沿，负脉冲的后沿是上升沿）

使用 741937419374193 芯片实现一个 303030 秒倒计时路，电路有一个频率为 444 Hz 的输入脉冲信号，一个倒计时启动信号 xxx，一个输出信号 ZZZ，当倒计时结束时输出 Z=1Z=1Z=1 且停止倒计时。

  • 因为倒计时需要的是 1Hz1Hz1Hz 的输入频率，所以需要分频器

    因为一个 TTT 触发器可以实现 0、10、10、1 状态，实现模 222 计数

    所以两个 TTT 触发器串联可以实现 00,01,10,1100,01,10,1100,01,10,11 4 种状态，实现模 444 状态

    2 0 1（下降沿触发）
    3 1 1
    4 0 0
