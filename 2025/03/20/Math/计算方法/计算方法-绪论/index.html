# 有效数字

有效数字的定义：若近似数 x∗x^*x∗ 的绝对误差限不大于 0.5×10−j0.5 \times 10^{-j}0.5×10−j，则称近似数 x∗x^*x∗ 准确到小数点后 jjj 位；从这个小数点后的第 jjj 位数字直到最左边非 0 数字之间的所有数字都称为 x∗x^*x∗ 的有效数字

如果近似值的有效位数是由四舍五入产生的，那么直接计算到第一个非零数有几位即可

如果将 xxx 的近似值 x∗x^*x∗ 表示成

x∗=±0.a1a2...an×10mx^*=\pm0.a_1a_2...a_n\times 10^m x∗=±0.a1​a2​...an​×10m

其中，a1,a2,...,ana_1,a_2,...,a_na1​,a2​,...,an​ 是 0∼90\sim 90∼9 之间的自然数， a1≠0a_1\neq 0a1​=0 ，那么若

则根据有效数字的定义可以推出 x∗x^*x∗ 有 lll 位有效数字

# 加减法的误差限

和的误差是误差之和，差的误差是误差之差

任意多个数的和或差的误差限等于各数误差限之和

以减法为例证明：

∣ε(x∗−y∗)∣≤∣ε(x∗)−ε(y∗)∣≤∣ε(x∗)∣+∣ε(y∗)∣≤η(x∗)+η(y∗)|\varepsilon(x^*-y^*)|\leq |\varepsilon(x^*)-\varepsilon(y^*)|\leq |\varepsilon(x^*)|+|\varepsilon(y^*)|\leq \eta(x^*)+\eta(y^*) ∣ε(x∗−y∗)∣≤∣ε(x∗)−ε(y∗)∣≤∣ε(x∗)∣+∣ε(y∗)∣≤η(x∗)+η(y∗)

# 近似计算遵循的原则

中间的计算结果应比上述各法则所提及的位数多取一位

# 加减运算

把小数位数多的四舍五入，使其比小数位数最少的数多一位小数

计算结果保留的小数位数和原近似数中小数位数最少者相同

# 乘除运算

各因子保留的位数应比有效数字位数最少者的位置多一位

结果的有效数字位数与原近似值中有效数字位数最少者的位数至多少一位

# 乘方与开方运算

原来近似值有几位有效数字，计算结果仍保留几位有效数字

# 对数运算

所取对数的位数应与其真数的有效数字的位数相等

# 注意的问题

  1. 避免两个相近的数相减
  2. 避免除数绝对值远小于被除数绝对值的除法
  3. 防止大数吃小数
  4. 尽量减少运算次数
