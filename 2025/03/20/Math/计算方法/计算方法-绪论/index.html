# 有效数字

若近似数 x∗x^*x∗ 的绝对误差限不大于 0.5×10−j0.5 \times 10^{-j}0.5×10−j，则称近似数 x∗x^*x∗ 准确到小数点后 jjj 位；从这个小数点后的第 jjj 位数字直到最左边非 0 数字之间的所有数字都称为 x∗x^*x∗ 的有效数字

如果近似值的有效位数是由四舍五入产生的，那么直接计算到第一个非零数有几位即可
