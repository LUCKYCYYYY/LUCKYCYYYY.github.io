# 1007 分配宝藏

1007 分配宝藏

题目描述：

船长拥有∞\infty∞ 宝藏，要求分给士兵们，确定好分配方案后士兵们会投票，如果大于等于半数的人（包括船长）对分配认可，那么可按此分配；否则，船长被处死，第一继承人上位。问如何分配船长能在分出最少的情况下活下来。

解题思路：（nnn 表示士兵的个数）

n = 1 ： 船长的 1 票已经过半，该士兵可给可不给，因为要求分出最少，所以给 0 。

n = 2 ： 第一顺位一定投反对，因为如果 Ta 成为船长就可以拥有所有宝藏；因为赞同票需要过半，对于第二顺位，如果这一票 Ta 不投赞成，那么在下一轮 Ta 无法获得宝藏，所以只需给 Ta 1 个宝藏，Ta 就会获得比下一轮多的宝藏，Ta 就会投赞同。

n = 3 ： 第一顺位一定投反对，因为如果 Ta 成为船长就可以拥有所有宝藏；第二顺位如果到下一轮将获得 0 宝藏，所以只需给 Ta 1 个宝藏，Ta 就会投赞同；因为包括船长自己票数过半，所以第三顺位可给可不给， 因为要求给出最少，所以给 0。

n = 4 ：第一顺位一定投反对，因为如果 Ta 成为船长就可以拥有所有宝藏；第二顺位如果到下一轮将获得 0 宝藏，所以只需给 Ta 1 个宝藏，Ta 就会投赞同；第三顺位如果到下一轮将获得 1 宝藏，第四顺位如果到下一轮将获得 0 宝藏，显然拉拢第四顺位所需宝藏更少，所给第三顺位 0 宝藏， 第四顺位 1 宝藏。

# 1002 船长

1002 船长

题目描述：个人淘汰赛，钦定特定某个人是赢家，除此之外其他人获胜率均五五开，按编号从小到大两个两个打，多出来的直接晋级，求赢家不碰上另外特定 kkk 个人的概率。

解题思路：

以钦定赢家 www 不想碰到的 kkk 个人为叶子节点建立二叉树，每往上走一层就是打了一轮。

为了方便给非叶子节点编号，我们将1∼n1\sim n1∼n 的编号都−1-1−1，那么两个节点x,yx, yx,y 的父节点就直接是x/2,y/2x / 2,y / 2x/2,y/2 ，并且两个打比赛的节点可以通过 x ^ 1 = y 得到。

叶子节点的概率 f[i]f[i]f[i] 初始化为 1，f[i]f[i]f[i] 表示 www 不想碰到的人到达 iii 这个位置的概率。

对于向上比赛，概率的计算可以分成四种情况：

  1. 如果节点 a[i]a[i]a[i] 和钦定赢家 www 打比赛，那么就直接计算不会碰到该节点的概率 ans *= (1 - f[i])
  2. 如果节点 (a[i] >> 1) == (a[i + 1] >> 1) ，即跟 a[i]a[i]a[i] 打比赛的节点也在 aaa 中，二者都是 www 不想碰到的人，那么 f[i >> 1] = (f[i] + f[i + 1]) / 2
  3. 如果节点 (a[i] ^ 1) != a[i + 1] && (a[i] ^ 1) < n ，即跟 a[i]a[i]a[i] 打比赛的节点不在 aaa 中，另一个节点是和 www 打必输的节点， 那么 f[i >> 1] = f[i] / 2
  4. 如果节点 (a[i] ^ 1) >= n ， 即 a[i]a[i]a[i] 是最后一个单出来的节点，不需要打就晋级，那么 f[i >> 1] = f[i]

小数取模：

注意：不能先用 doubledoubledouble 再取模，因为 doubledoubledouble 涉及精度不够的问题

因为最后的结果要求对 998244353 取模，而 ans 又是小数，所以需要利用乘法逆元

如果我们所有的除操作都用乘法逆元那么会 TLE，而我们发现所有的除法运算都是 / 2 所以只需要算 222 的乘法逆元即可，一定要预处理 2 的乘法逆元

求 222 的逆元有两种方式：

  1. 乘法逆元的定义:

    a×a−1≡1(modm)a \times a^{-1} \equiv 1\pmod m a×a−1≡1(modm)

    对于 a=2a = 2a=2， 即找到 xxx 使得

    2×x≡1(mod998244353)2 \times x\equiv 1\pmod {998244353} 2×x≡1(mod998244353)

    等价于

    2x=998244353k+12x = 998244353k + 1 2x=998244353k+1

    经过尝试， 因为 xxx 为整数， 所以 kkk 取奇数， 发现 k=1k = 1k=1 时即可

    得到结论：对于任意奇数模数 mmm，2 的乘法逆元可以通过以下公式计算：

    inv2=m+12inv2 = \frac{m + 1}{2} inv2=2m+1​

  2. 直接利用求乘法逆元的公式，利用快速幂求逆元

    ab≡a×x(modm)\frac{a}{b}\equiv a\times x\pmod m ba​≡a×x(modm)

    ⇒x=am−2mod  m\Rightarrow x = a ^ {m - 2}\mod m ⇒x=am−2modm

# 1004 海浪

1004 海浪

题目表述：

题目给定一个长度为 nnn 的数组 aaa，定义一段区间 [l,r][l,r][l,r] 为 “波浪” 区间，当区间内存在一个 “波峰” 或 “波谷”，使得整个区间内的元素都围绕这个波峰或波谷波动。每次查询给出一个区间 [l,r][l,r][l,r] ，要求在这个区间内找到最长的 “波浪” 子区间。

解题思路：

海浪一共只有两种形式

  1. 奇数位为 “波峰”， 偶数位为” 波谷 “。即如果 [l,r][l, r][l,r] 区间中的波面是海浪，那么 [l,r][l, r][l,r] 中奇数位的最小值 大于 偶数位的最大值。
  2. 奇数位为 “波谷”， 偶数位为 “波峰”。即如果 [l,r][l, r][l,r] 区间中的波面是海浪，那么 [l,r][l, r][l,r] 中奇数位的最大值 小于 偶数位的最小值。

显然我们只需要处理一种情况，对于另一种情况只需要对距离取反，再以相同的方式处理一遍即可。

实现过程：（以第一种情况为例）

  1. 初始化：既然要维护奇数位的最小值和偶数位的最大值，那么就可以利用 ST 表。

    很明确，在我们要求奇数位的最小值的时候偶数位不能在里面，那么我们就需要把偶数位的最小值初始化为+∞+\infty+∞；在我们要求偶数位的最大值的时候奇数位不能在里面，所以需要把奇数位的最大值初始化为−∞-\infty−∞

  2. 判断一个区间是否是海浪：利用 ST 表，我们可以判断，如果一个区间奇数位的最小值 大于 偶数位的最大值， 那么这个区间是海浪。

    需要注意：对于只有一个节点的情况，假设该节点是奇数位，那么因为我们将该点的 maxmaxmax 初始化为 −∞-\infty−∞， minminmin 初始化为 +∞+\infty+∞ ，所以只有一个点的时候这个点也是海浪

  3. 计算最长波浪距离和节点：利用双指针，我们可以得到以 iii 为左端点，波浪的最远距离len[i][0]len[i][0]len[i][0]，距离最远时的右端点是pos[i]=jpos[i] = jpos[i]=j

    • len[i][j]len[i][j]len[i][j] 表示从 iii 开始，2j2 ^ j2j 距离的节点中任一节点，以该节点为左端点的波浪区间的长度最大值
    • pos[i]pos[i]pos[i] 表示以 iii 节点为左端点， 最长波浪的右端点；很显然，pospospos 是的单调递增的

  4. 处理询问：对于每一组给定的 [l,r][l, r][l,r]，我们可以将总区间分成三段：在 pospospos 数组中左端点 iii 在 lll 左边， 在 lll 和 rrr 中间， 以及在 rrr 右边。具体如下图，其中黑色线段表示 pospospos 数组，每个线段的左端点表示 pospospos 数组中的左端点 iii， 右端点表示 pospospos 数组中的值 jjj， 即以 iii 为左端点的海浪距离最长时的右端点。

    • 若左端点不在查询区间 [l,r][l, r][l,r]
      • 在 111 区：因为 pospospos 数组是递增的，所以这显然不如以lll 为左端点，因为pos[l]pos[l]pos[l] 一定比pos[i](i<l)pos[i](i<l)pos[i](i<l) 的值靠右

      • 右端点在查询区间 j<=rj <= rj<=r， 利用 ST 表，利用 lenlenlen 找到最长的波浪
