# 最小项

一个具有 nnn 个变量的函数的与项包含 nnn 个变量，每个变量都以原变量或反变量形式出现且仅出现一次

性质：

  • nnn 个变量可以构成 2n2^n2n 个最小项
  • 任意一个最小项，其相应变量有且仅有一种取值使这个最小项的值为 1
  • 相同变量构成的两个不同最小项相与为 0
  • nnn 个变量的全部最小项相或为 1
  • nnn 个变量构成的最小项有 nnn 个相邻最小项（相邻最小项：除一个变量互为相反外其余变量均相同的最小项）

表示方式：

  • nnn 变量的最小项用 minm_i^nmin​ 表示，简写成 mim_imi​
  • 下标 iii 的取值规则：原变量用 1 表示，反变量用 0 表示，得到的二进制数就是 iii 的值

# 最大项

一个具有 nnn 个变量的函数的或项包含 nnn 个变量，每个变量都以原变量或反变量形式出现且仅出现一次

性质：

  • nnn 个变量可以构成 2n2^n2n 个最大项
  • 任意一个最大项，其相应变量有且仅有一种取值使这个最大项的值为 0
  • 相同变量构成的两个不同最大项相或为 1
  • nnn 个变量的全部最大项相与为 0
  • nnn 个变量构成的最大项有 nnn 个相邻最大项（相邻最大项：除一个变量互为相反外其余变量均相同的最大项）

表示方式：

# 卡诺图化简法

# 构成

nnn 变量的卡诺图是由 2n2^n2n 个小方格构成的平面方格图，每一个小方格与一个最小项对应，方格中相邻两个方格互为相邻最小项

相邻分为：

  • 几何相邻：指两个小方格在几何位置上是相邻的
  • 相对相邻：同一行的两端或者同一列两端的两个小方格所代表的最小项也是相邻最小项
  • 重叠相邻：当变量数大于 4 时，对于变量 ABCDE 的情况，需要将 A = 1 的卡诺图看成重叠在 A = 0 的卡诺图之上，处于重叠位置的小方格也是相邻最小项

# 最小项合并规律

两个相邻最小项可以合并为一项并消去一个变量

卡诺图中相邻的 2m2^m2m 个小方格含有 mmm 个不同变量，n−mn-mn−m 个相同变量，合并后可以消去 mmm 个变量，卡诺图对应的最小项可用 n−mn-mn−m 个变量的与项表示

  1. 2m2^m2m 个小方格组成一个大格（可以是长方形、也可以是正方形）

  1. 2m2^m2m 个小方格处在一行（一列）

  1. 2m2^m2m 个小方格处在两个边行（边列）

  1. 2m2^m2m 个小方格处在 4 个边角

  1. 单独一个方格

  1. 整个卡诺图

归纳：

  • 卡诺图中小方格的个数必须为 2m2^m2m 个，m≤nm\leq nm≤n
  • 卡诺图中的 2m2^m2m 个小方格有一定的排列规律，它们含有 mmm 个不同变量，n−mn-mn−m 个相同变量

  • 卡诺圈尽量少⇒\Rightarrow⇒ 化简结果中与项个数少
  • 允许重复圈小方格，但每个卡诺圈至少有一个新的小方格

# 最简或与表达式

  • 当给定逻辑函数为一般与或表达式
    1. 做出 FFF 的卡诺图
    2. 合并卡诺图上的 0 方格
    3. 求出反函数 Fˉ\bar FFˉ 的最简与或表达式
    4. 对 Fˉ\bar FFˉ 的最简与或表达式取反
  • 当给定逻辑函数为或与表达式（两次取反法）
    1. 求出反函数 Fˉ\bar FFˉ，并做出反函数的卡诺图
    2. 合并卡诺图上的 1 方格
    3. 求出反函数 Fˉ\bar FFˉ 的最简与或表达式
    4. 对 Fˉ\bar FFˉ 的最简与或表达式取反
  • 当给定逻辑函数为或与表达式（两次对偶法）
    1. 求出 FFF 的对偶函数 F′F'F′，并做出对偶函数 F′F'F′ 的卡诺图
    2. 合并卡诺图上的 1 方格
    3. 求出对偶函数 F′F'F′ 的最简与或表达式
    4. 对 F′F'F′ 的最简与或表达式取对偶
