# 常用的代码

# BCD 码

# 8421 码

从高位到低位的权值依次为 8、4、2、18、4、2、18、4、2、1

1010∼11111010\sim 11111010∼1111 为无效码

# 2421 码

从高位到低位的权值依次为 2、4、2、12、4、2、12、4、2、1，将任意一个十进制数码 DDD 的242124212421 码各位取反，得到的 242124212421 码正好是 DDD 对 999 的补码

0101∼10100101\sim 10100101∼1010 为无效码

# 5421 码

从高位到低位的权值依次为 5、4、2、15、4、2、15、4、2、1

0101,0110,0111,1101,1110,11110101,0110,0111,1101,1110,11110101,0110,0111,1101,1110,1111 为无效编码

# 余三码

无权码

余 3 码的每一个编码比 842184218421 多 3(0011)3(0011)3(0011)

0000、0001、0010、1101、1101、11110000、0001、0010、1101、1101、11110000、0001、0010、1101、1101、1111 为无效编码

和 242124212421 码类似，是对 999 的自补代码

在两个余 333 码表示的十进制数相加时，需要修正：如果有进位，则结果加 333；如果结果无进位，则结果减 333
