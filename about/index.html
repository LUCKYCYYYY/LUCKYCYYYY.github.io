# 😆关于 “眨眼的小星玥”

网站名 “眨眼的小星玥” 其实是从原来的名字 “眨眼的小星星” 加上自己名字中的 “玥” 迁移过来的

网站原本是 butterfly 主题，但是奈何 shoka 主题的精选分类反转功能深得我心，遂熬了两个大夜把主题给改了。

但是，因为之前 butterfly 主题是支持图片在本地然后上传上去，但是我在尝试了 shoka 主题的本地图片上传功能后，发现图片无法正常显示（待有空再试试），所以部分图片需要重新上传图床（˃̣̣̥᷄⌓˂̣̣̥᷅）

网站内容大致会分为以下几类：

  1. 课内专业课的学习知识点总结
  2. 算法知识的学习
  3. 参加竞赛的准备工作
  4. 一些杂杂的知识
