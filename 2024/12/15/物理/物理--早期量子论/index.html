# 康普顿效应

# 实验规律

  1. 散射的射线中有与波长λ0\lambda_0λ0​ 相同的射线，也有波长λ>λ0\lambda>\lambda_0λ>λ0​ 的射线
  2. 散射线波长的该变量Δλ=λ−λ0\Delta\lambda=\lambda-\lambda_0Δλ=λ−λ0​ 随散射角θ\thetaθ 的增加而增加Δλ=λc(1−cos⁡θ),λc=2.4263×10−12m\Delta\lambda=\lambda_c(1-\cos\theta),\lambda_c=2.4263\times10^{-12}mΔλ=λc​(1−cosθ),λc​=2.4263×10−12m
  3. 同一散射角下波长的改变量相同，与散射物质无关
  4. 原子量较小的物质，散射较强

# 实验解释

λ>λ0\lambda>\lambda_0λ>λ0​：由于物质中的外层电子的动能远小于入射光子的动能，碰撞前电子可以看作是静止的。碰撞过程中，光子的一部分能量传递给电子，能量减小，频率减小，因而波长增大。

原子量较小的物质，电子束缚很弱⇒\Rightarrow⇒ 康普顿散射较强
