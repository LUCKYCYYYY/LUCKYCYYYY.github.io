# 第一型曲线积分

# 形式

∫Lfds\int_{L}f ds∫L​fds​ – 表示曲线 L 的质量，f 是密度函数

# 方法

  1. 化为定积分

∫Lfds=∫αβf(x(t),y(t),z(t))x(t)′2+y(t)′2+z(t)′2dt\int_{L}f ds = \int_{\alpha}^{\beta}f(x(t),y(t),z(t))\sqrt{x(t)'^{2} + y(t)'^{2} +z(t)'^{2}}dt∫L​fds=∫αβ​f(x(t),y(t),z(t))x(t)′2+y(t)′2+z(t)′2​dt​

  • 极坐标：ds=r(θ)2+r(θ)′2dθds=\sqrt{r(\theta)^2 + r(\theta)'^2}d\thetads=r(θ)2+r(θ)′2​dθ
  • xy 平面曲线：ds=1+y(t)′2dxds = \sqrt{1+y(t)'^2}dxds=1+y(t)′2​dx​
  1. 对称性
  • 平面曲线 L 关于y=xy=xy=x 对称，且被积函数在 L 上连续：可以交换 x，y
  • 空间曲线 L 关于y=x=zy=x=zy=x=z 对称，且被积函数在 L 上连续：x,y,z 具有轮换对称性

  1. 质心

    当被积函数是只有一次参变量时，可以利用已知质心进行计算

∫LF⃗⋅dr⃗=∫LF⃗⋅τ⃗ds=∫LPdx+Qdy+Rdz=∫LPcosα+Qcosβ+Rcosγds\int_{L}\vec{F} \cdot d\vec{r} = \int_L \vec{F} \cdot \vec{\tau}ds = \int_LPdx+Qdy+Rdz = \int_L Pcos\alpha + Qcos\beta + Rcos\gamma ds∫L​F⋅dr=∫L​F⋅τds=∫L​Pdx+Qdy+Rdz=∫L​Pcosα+Qcosβ+Rcosγds

# 方法

  1. 化为定积分

  2. Green 公式

    \oint_L Pdx + Qdy = \iint_\limits{D}(Q_x - P_y)dxdy

    • 简单闭曲线
    • 函数 P，Q 在 D 上包括边界有连续一阶偏导数

  3. 积分与路径无关

  • 可化成全微分
  • Qx=PyQ_x = P_yQx​=Py​
  • D 内任何分段光滑闭曲线 C，∮CPdx+Qdy=0\oint_C Pdx+Qdy = 0∮C​Pdx+Qdy=0
  1. 除M0M_0M0​ 点外，P，Q 处处有一节连续偏导数，Qx=PyQ_x = P_yQx​=Py​，任何\color{red}{任何}任何 包围M0M_0M0​ 的正向\color{red}{正向}正向 简单闭曲线 L，积分∮LPdx+Qdy\oint_L Pdx+Qdy∮L​Pdx+Qdy 取同一值
